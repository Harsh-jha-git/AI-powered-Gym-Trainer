PS4 — AI-Based Human Action Analysis Using Video Domain: 

Computer Vision / Video Analytics Problem Statement Create an AI system that analyzes human 
activity recognition from video input to evaluate movement quality, coordination, and performance. 

The system may be applied to sports training, rehabilitation, or skill assessment use cases. 

Key Objectives: 
    Perform human pose estimation from video 
    Analyze joint angles and motion patterns 
    Compare movements against reference actions 
    Provide quantitative and visual feedback 

Datasets / Resources 
Resource Link:
    UCF101 Action Recognition Dataset https://www.crcv.ucf.edu/data/UCF101.php 
    
Expected Output:
    Annotated video output with pose overlay 
    Skill score or performance metrics 
    Visual and textual feedback
